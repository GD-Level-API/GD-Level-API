GD Level API

Datos de niveles de Geometry Dash + card PNG generada.
Lista para usar en bots de Discord, webs y apps.

https://thumball.liamt.xyz

Endpoints

GET /api/level?id={levelId} ▼
Devuelve un JSON con todos los datos del nivel: nombre, creador, estadísticas, canción, thumbnail y diff face ya resueltos.
id* ID numérico del nivel en Geometry Dash
curl

Python

console.log(level.name);       // "1st level"
console.log(level.difficulty); // "Hard"
console.log(level.urls.card);    // URL de la card PNG

GET /api/levels?ids={id1,id2,...} ▼
Devuelve un array JSON con datos de múltiples niveles en una sola request. Máximo 10 IDs por request.
ids* IDs separados por coma — ej: ?ids=128,1,2
curl
JavaScript
Python
curl "https://thumball.liamt.xyz/api/levels?ids=128,1,2"
const res    = await fetch("https://thumball.liamt.xyz/api/levels?ids=128,1,2");
const levels = await res.json(); // array de niveles

levels.forEach(l => console.log(l.name, l.difficulty));
import requests

GET /api/card?id={levelId} ▼
Devuelve una imagen PNG con la card del nivel — thumbnail, ícono de dificultad, estadísticas y badges. La primera request genera la imagen (~1s), las siguientes son cacheadas y son instantáneas.
id* ID numérico del nivel
size opcional normal (1600×520px, por defecto) · small (1200×320px)

const { EmbedBuilder, AttachmentBuilder } = require('discord.js');

// Slash command handler
async execute(interaction) {
  const id = interaction.options.getInteger('id');

  // Cargar datos y card en paralelo
  const [data] = await Promise.all([
    fetch(`https://thumball.liamt.xyz/api/level?id=${id}`).then(r => r.json()),
  ]);

  if (data.error) return interaction.reply({ content: 'Nivel no encontrado', ephemeral: true });

  const card  = new AttachmentBuilder(data.urls.card, { name: 'card.png' });
  const embed = new EmbedBuilder()
    .setTitle(data.name)
    .setDescription(`por **${data.author}** · ${data.difficulty}`)
    .setImage('attachment://card.png')
    .setColor(0xf0c94e);

  await interaction.reply({ embeds: [embed], files: [card] });
}
import discord
from discord import app_commands
import aiohttp
import io

@tree.command(name="nivel", description="Info de un nivel GD")
async def nivel(interaction: discord.Interaction, id: int):
    async with aiohttp.ClientSession() as s:
        # Datos y card en paralelo
        async with s.get(f"https://thumball.liamt.xyz/api/level?id={id}") as r:
            data = await r.json()
        async with s.get(data["urls"]["card"]) as r:
            png = await r.read()

    file  = discord.File(io.BytesIO(png), filename="card.png")
    embed = discord.Embed(
        title=data["name"],
        description=f"por **{data['author']}** · {data['difficulty']}",
        color=0xf0c94e
    ).set_image(url="attachment://card.png")
    await interaction.response.send_message(embed=embed, file=file)
