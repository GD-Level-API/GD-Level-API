⭐ GD Level API

Datos de niveles de Geometry Dash + thumbnail + card PNG generada — todo en un endpoint.

GET /api/level?id={levelId}
Devuelve un JSON con todos los datos del nivel ya resueltos: nombre, creador, estadísticas, thumbnail y URL de la diff face.
id ID numérico del nivel en Geometry Dash (requerido)

GET /api/card?id={levelId}
Devuelve directamente una imagen PNG (800×260px) con la card del nivel. Podés embeber la URL en una web con <img src="/api/card?id=128"> o enviarla como attachment en un bot.
id ID numérico del nivel en Geometry Dash (requerido)
